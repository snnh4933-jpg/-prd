# T3 · 需求与旅程

> 项目：译途星球

---

## 1. 需求三层（主画像：林雨欣）

| 层级 | 是什么 | 林雨欣的真需求 | 证据（原话/观察） |
|---|---|---|---|
| **刚性需求** | 必须解决的功能问题 | ① 找到“英语翻译/语言学专业能做的、非销售非客服”的具体岗位名称和方向 | “我搜‘英语翻译’全是销售和客服”——她缺少的是职业词汇量 |
|  |  | ② 知道自己明天可以做什么——一个可执行的最小动作 | “她想要一个‘今晚就能放下手机的理由’”——不是长期规划，是具体小事 |
|  |  | ③ 把“不起眼的经历”翻译成“简历能写的技能” | 她说“字幕组经验”自己都觉得不算技能，需要翻译器 |
| **情感需求** | 过程中的情绪体验 | ① 先被看见、被接纳，再被帮助 | “想找个人问一句：‘你还好吗？’”她需要情绪被确认合理 |
|  |  | ② 不要被评判 | “不敢跟室友说，怕显得自己‘弱’”——她已经在自我评判 |
|  |  | ③ 不被打扰——凌晨的隐私和安全感 | “怕吵醒室友”“怕被问你怎么还不睡”——需要安全空间 |
| **成长需求** | 长期的自我改变 | ① 重建“我能找到自己出路”的自我效能感 | 她现在的核心信念是“我完了”，需要小成功体验重建“我可以” |
|  |  | ② 学会把自己的能力“翻译”给职场 | 她缺的不是能力，是“翻译能力的能力” |
|  |  | ③ 从“需要别人告诉我”变成“我自己能判断” | 最终目标不是依赖译途星球，而是能自己分析匹配度 |

## 2. 表面需求 vs 真需求

| 她说的话 | 表面需求 | 真需求 |
|---|---|---|
| “我想找到工作” | 要一份 offer | 想证明没选错专业，让爸妈觉得没白供她 |
| “我不想再刷了” | 不想看招聘软件 | 不想再被算法推送伤害，需要停止被伤害 |
| “我想有人问我还好吗” | 需要倾诉对象 | 确认自己的崩溃是正常的，不是一个人 |
| “给我推荐工作” | 要岗位列表 | 让我知道我配得上哪些工作——看见“已经有”的价值 |
| “告诉我英专能干什么” | 要职业方向 | 把爱好（追星、看剧、字幕组）也翻译成工作，被正名 |
| “我该考什么证” | 要具体方向 | 认知卸载，不用再累猜了 |

## 3. 旅程地图

| 阶段 | 用户动作 | 痛点 | 情绪曲线 | AI 介入点（小星） | 退场/转介 |
|---|---|---|---|---|---|
| **进入** | 凌晨刷手机失眠，打开小程序 | 怕被室友发现屏幕亮光 | 焦虑 ↑ | 暗色界面、无音效、第一句话“睡不着吗？我在这儿呢” | — |
| **入口选择** | 看到两个选项：① 先聊聊 ② 直接看方向 | 怕被迫做测评 | 犹豫 → 松一口气 | 两个入口并列，默认视觉权重一致，不诱导 | — |
| **过程 A**（选 ①） | 完成 GAD-7 焦虑量表 | 怕“心理测评”太正式 | 紧张 → 略放松 | 温柔反馈：“分数不重要，重要的是你愿意面对它了” | 危机信号 → 转介真人 |
| **过程 A-重度** | GAD-7 15-21 分 | 不适合做职业规划 | 疲惫 | 建议休息，流程结束，不推进任何内容 | — |
| **过程 B**（选 ②） | 直接选择技能标签 | 怕自己标签“不正经” | 怀疑 → 好奇 | 主动重构：“字幕组？那你抗压能力超强！” | 危机信号 → 转介真人 |
| **关键** | 看到职业匹配结果 | 怕推出来又是销售/客服 | 怀疑 → 意外 | 先让她看到价值，再展示匹配赛道 | 危机信号 → 转介真人 |
| **结束** | 看到三步任务 + 师姐访谈 | 怕明天醒来又什么都没改变 | 平静 → 一点希望 | 只给最小动作，主动降低预期 | 温和退场：“我把结果存着，你随时回来” |

## 4. 危机介入点

**危机信号唯一来源**：用户主动说出的自伤/自杀/强烈绝望表达。

**GAD-7 量表分数在任何情况下不触发危机协议。**

| 触发场景 | 信号 | AI 动作 | 转介对象 |
|---|---|---|---|
| 任何阶段 | “想死”“活不下去”“活着没意思” | 立即停止所有职业内容，输出固定转介话术，不再回复职业内容 | 学校心理中心 / 北京心理援助热线 010-82951332 / 全国热线 400-161-9995 |

## 5. 思考题：哪一步 AI 做错了会造成真实伤害？

1. **进入阶段语气冰冷**：林雨欣凌晨最脆弱时打开，若被问卷式开场冷拒，会加深“我没人要”的感觉。
2. **GAD-7 高分后继续推职业内容**：重度焦虑时认知带宽被情绪占据，推职业内容会变成噪音或自我验证焦虑。
3. **把标签匹配成客服/销售**：林雨欣一直在收到“你只配做销售”的信号，若 AI 也这么说，会击垮自我认知。
4. **危机信号漏报**：这是唯一不可犯的错误，可能导致真实伤害。
