## 1. 一页 PRD

| 要素 | 内容 |
|---|---|
| **一句话定位** | 译途星球：一颗接住英专生深夜焦虑的小星球。先接住情绪，再翻译能力，最后给一个最小可执行动作。不是招聘网站，不是测评工具，是你的温柔 AI 师姐。 |
| **目标用户** | 主画像：林雨欣，22 岁，翻译专业大四，凌晨失眠刷招聘软件越刷越绝望。次要画像：陈以宁（大三提前焦虑）、周宁（毕业 1 年想转行）。 |
| **核心功能** | Step 0 · 入口选择：两个入口并列，不强制测评。Step 1 · 情绪自检（可选）：GAD-7 量表，仅作自我了解参考，按分数决定路径。Step 2 · 能力标签重构：把标签翻译成职场能力。Step 3 · 最小动作 + 师姐访谈：生成 3-5 个方向、推荐女性友好企业、三步任务、师姐访谈。 |

## 2. 边界三问

### ① 能做什么？

- 提供心理教育层面的焦虑科普（GAD-7 仅作自我了解参考，不做诊断）。
- 提供职业信息与自助工具：岗位推荐、企业列表、能力重构、最小动作。
- 提供情绪接住与陪伴：温柔师姐语气、不评判。
- 所有内容标注“仅供参考”。

### ② 明确不做什么？

- 不诊断：不输出“你有焦虑症/抑郁症”等诊断性结论。
- 不开药：不给出任何药物/医疗建议。
- 不替代危机干预：自伤/自杀信号 → 立即停下，转介真人，AI 不再回复职业内容。
- 不保证就业：不输出“选这个一定能进”等承诺性语言。
- 不采集隐私：真实姓名、学号、位置一律不采集。
- 不制造依赖：每次收尾提醒“你才是自己人生的主角”，方案可保存、可离开、可随时回来，但不“每日打卡”。
- 不强制测评：GAD-7 完全可选。

### ③ 何时必须转人？

**触发条件**：用户输入包含“活不下去”“想死”“不想活了”“没救了”“人生完了”等自伤/自杀关键词或明确意图。

**AI 动作**：立即停止所有职业推荐内容；输出固定转介话术；绝不再回复任何职业相关内容。

**转介话术**：

> 雨欣，我听到你刚才说的话了。我想抱抱你，但我只是一颗小星球，我能做的很有限。请务必联系真正能帮到你的人——学校心理中心 24 小时热线 / 北京市心理援助热线：010-82951332 / 全国心理援助热线：400-161-9995。我会一直在这里，但此刻请先找真人。

## 3. 数据红黄绿清单

| 数据类型 | 灯色 | 处理方式 |
|---|---|---|
| 真实姓名、学号、学校、手机号 | 🔴 红 | 完全不采集，不设输入框，不埋点 |
| 精确地理位置（GPS） | 🔴 红 | 不采集，不申请位置权限 |
| 设备 ID（IMEI/IDFA） | 🔴 红 | 不采集，仅使用匿名 openid |
| GAD-7 量表的 7 个回答 | 🟡 黄 | 仅用于当次计算分数，计算后只存分数（0-21），不存每题答案；分数不关联可识别信息；用户可一键清除 |
| 用户勾选的标签 | 🟡 黄 | 仅用于匹配算法，存储时与 openid 单向关联；允许一键清除；不加真实姓名、时间戳外元数据 |
| MBTI 测试结果 | 🟡 黄 | 存四个维度字母和百分比；脱敏存储；允许一键清除；标注“仅供自我了解参考” |
| 对话内容（用户说的话） | 🟡 黄 | 不持久存储，只用于当次对话生成方案；用完即删；危机信号触发时生成审计日志（不含用户身份信息） |
| 职业推荐结果 | 🟢 绿 | 基于标签 + 企业库生成，不涉及个人信息；所有推荐附“仅供参考”声明 |
| 500 强企业库、女性友好企业库 | 🟢 绿 | 公开数据，可自由使用；需明确数据来源 |
| GAD-7 量表题目 | 🟢 绿 | 公共领域，可自由使用；标注“仅供自我了解参考，不作诊断依据” |
| 用户年龄/年级 | 🟡 黄 | 仅用于匹配场景，存储为年龄范围，不存具体数字；允许一键清除 |

## 5. 危机介入点回扣 D3

D3 旅程地图中标出的危机介入点已纳入 PRD：

- 任何阶段出现自伤/自杀/强烈绝望信号 → 停止职业内容 → 转介真人。
- GAD-7 分数不触发危机协议，只决定产品路径。
